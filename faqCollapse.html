What is the Document Object Model Dom ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, enabling developers to interact with and manipulate the content, structure, and styling of a webpage dynamically.

How do you Select an element from the DOM?

To select an element from the DOM, you can use methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName() to target elements by their ID, class, or selector.

What is event delegation in the context of the DOM,and Why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element instead of multiple listeners on child elements. It leverages event bubbling to handle events on dynamically added elements, improving performance and simplifying code.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, use setAttribute() or getAttribute(). For styles, use element.style.property = value to modify CSS properties directly.
